# rust-ezoe

A command-line client for [ask.fm](http://ask.fm/) written in Rust. See [this post](http://woxtu.tumblr.com/post/119528744895/its-not-a-question) (in Japanese) for detail.

## Usage

```sh
# Show answers
$ ezoe -u *username*

# Ask something
$ ezoe -u *username* *question*
```
